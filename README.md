# Amdox Task Management System
A full-stack task management web application built using Java, Spring Boot, Hibernate, MySQL, and React. The platform helps teams organize tasks, track progress, collaborate in real time, and improve productivity.

# Features
* User Authentication & Authorization
* Task Assignment and Prioritization
* Role-Based Access Control
* Deadline Tracking & Notifications
* Real-Time Collaboration
* Progress Reporting Dashboard
* Secure Login System
* Responsive UI Design

## Installation

### Frontend Setup

### Backend Setup

1. Open backend project in IDE
2. Configure MySQL database
3. Run Spring Boot application
